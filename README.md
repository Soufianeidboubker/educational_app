# Educational App

A cross-platform Flutter application for educational purposes.

**Flutter Version**: 3.32.8  
**Dart Version**: 3.2.0  
**Status**: Active Development  

## Features
- Multi-platform support (Android, iOS, Web, Windows, macOS, Linux)
